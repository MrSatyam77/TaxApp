Please follow these instruction to backup your returns from TRX Software 2013.

Start and log into TRX software 2013.

At the upper-left corner on the screen, click on File.

From the File menu choose Backup Return.

After selecting Backup Return, the Backup to Disk Drive window will open up.

Select returns you would like to backup.

Please note down default backup location displayed in bottom or choose your preferred one.

On pressing of done, popup with message 'Backup completed successfully' will appear.

Create zip file of that backup.


Note

Maximum Zip file size: 3 GB
We recommend for you to create one zip file of your backup folder instead of zipping one return at a time. Do you have a Zip file bigger than 3 GB? Try preparing backups in batches, and upload each batch separately. Need assistance? Please review the Training Articles, or contact our Support Department.
Do you have large no.of returns to be converted? Try preparing backups in batches and upload each batch separately

Please follow the instructions below:

On the menu bar, click Tools, select File Maintenance, then Backup

Backup dialog box appears

Click the Select a backup location button, and select the drive and folder for your backup

Check only the box for Client Files

Click Backup

Create zip file of that backup.
